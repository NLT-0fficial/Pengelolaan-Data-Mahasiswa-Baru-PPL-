# ALUR SISTEM PENGELOLAAN DATA MAHASISWA PADA SISTEM INFORMASI AKADEMIK

## A. PROSES ADMIN MENGELOLA DATA MAHASISWA

### 1. ADMIN MEMBUKA SISTEM
- Admin mengakses halaman Sistem Informasi Akademik melalui browser.
- Sistem menampilkan halaman **Login**.

### 2. ADMIN MELAKUKAN LOGIN
- Admin memasukkan **username** dan **password**.
- Sistem melakukan verifikasi data login.
- Jika data benar, sistem menampilkan **Dashboard Admin**.
- Jika data salah, sistem menampilkan pesan gagal login.

### 3. ADMIN MELIHAT DASHBOARD
Sistem menampilkan informasi:
- Total mahasiswa
- Jumlah program studi

Admin dapat memilih menu yang tersedia.

### 4. ADMIN MEMBUKA MENU DATA MAHASISWA
- Admin memilih menu **Data Mahasiswa**.
- Sistem menampilkan tabel data mahasiswa yang berisi:

  - NIM
  - Nama
  - Program Studi
  - Semester
  - Email
  - Status

### 5. ADMIN MENAMBAH DATA MAHASISWA
- Admin menekan tombol **Tambah**.
- Sistem menampilkan form tambah mahasiswa.
- Sistem otomatis membuat **NIM** berdasarkan program studi yang dipilih.
- Admin mengisi data:

  - Nama
  - Email
  - Program Studi
  - Semester
  - Status

- Admin menekan tombol **Simpan**.
- Sistem menyimpan data ke database.
- Sistem menampilkan data mahasiswa terbaru.

### 6. ADMIN MENCARI DATA MAHASISWA
- Admin memasukkan kata kunci pada kolom pencarian.

Contoh:
- Nama mahasiswa
- NIM
- Program Studi

- Sistem menampilkan hasil pencarian yang sesuai.

### 7. ADMIN MENGUBAH DATA MAHASISWA
- Admin memilih tombol **Edit**.
- Sistem menampilkan form edit.
- Admin melakukan perubahan data.
- Admin menekan tombol **Update**.
- Sistem memperbarui data pada database.

### 8. ADMIN MENGHAPUS DATA MAHASISWA
- Admin menekan tombol **Hapus**.
- Sistem menampilkan konfirmasi penghapusan data.
- Admin menekan tombol **OK**.
- Sistem menghapus data dari database.

### 9. ADMIN MENCETAK LAPORAN
- Admin menekan tombol **Cetak**.
- Sistem menghasilkan laporan data mahasiswa.
- Laporan dapat dicetak atau disimpan.

## B. PROSES MAHASISWA MENGAKSES INFORMASI AKADEMIK

### 1. MAHASISWA MEMBUKA SISTEM
- Mahasiswa mengakses Sistem Informasi Akademik.

### 2. MAHASISWA MELAKUKAN LOGIN
- Mahasiswa memasukkan akun.
- Sistem memverifikasi data login.
- Sistem menampilkan halaman mahasiswa.

### 3. MAHASISWA MELIHAT INFORMASI AKADEMIK

Mahasiswa dapat melihat:

- NIM
- Nama
- Program Studi
- Semester
- Status Akademik

### 4. MAHASISWA MELAKUKAN PENCARIAN DATA
- Mahasiswa memasukkan kata kunci tertentu.
- Sistem menampilkan informasi yang dicari.

### 5. MAHASISWA SELESAI MENGGUNAKAN SISTEM
- Mahasiswa melakukan **Logout**.
- Sistem mengakhiri sesi pengguna.
